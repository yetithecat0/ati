# Contexto del Proyecto Antigravity
- Brand: ./brand/ATI_design_brief.html (Reglas core de marca)
- Stitch: ./stitch/ (Componentes y lógica)

## Reglas de Ejecución:
1. Siempre que se solicite un diseño o código, consulta primero `brand/ATI_design_brief.html` para validar colores, tipografías y tono de voz.
2. Para la implementación técnica, revisa la carpeta `stitch/` correspondiente a la funcionalidad solicitada.
3. Si hay conflicto entre una funcionalidad nueva y el `design_brief`, prioriza la consistencia de marca y notifícame el conflicto.
