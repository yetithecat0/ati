# ATI / Alphas Tool Interactive 🛡️✨

**ATI** es una estación de comando digital minimalista diseñada para centralizar ecosistemas de Inteligencia Artificial, herramientas Web y flujos de trabajo de alto rendimiento. Construida con una arquitectura de "Pads" táctiles y un enfoque en la eficiencia visual.

## 🚀 Características Principales v1.0

- **ADN de Sincronización:** Sistema de respaldo en Markdown con metadatos estructurales (Base64) para portabilidad total entre dispositivos.
- **ATI Cloud & Presets:** Galería de configuraciones maestras que permite cargar entornos pre-diseñados (Preset: **YETI THE CAT MASTER**).
- **Núcleo de Inteligencia:** Centro de ayuda y guía estratégica para el dominio de herramientas digitales.
- **Arquitectura de Mesas de Trabajo:** Organización jerárquica de espacios para diferentes facetas de desarrollo y gestión.
- **Modo Lanzamiento vs. Edición:** Interfaz dinámica para configuración fluida o ejecución sin distracciones.

## 🎨 Design System (Stitch)
El proyecto utiliza tokens de diseño propietarios:
- **Base:** Obsidian Dark Mode (#0e0e10, #16161a).
- **Accent:** Electric Purple (#7F77DD) & Indigo (#534AB7).
- **Typography:** Space Grotesk (Headings) & DM Sans (UI).

## 🛠️ Tecnologías
- **Next.js 15 (App Router)**
- **TypeScript**
- **Zustand** (Estado Atómico)
- **Tailwind CSS**
- **Material Symbols** (Iconografía)

## 📦 Instalación y Desarrollo
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`
3. Iniciar entorno de desarrollo: `npm run dev`
4. Construir para producción: `npm run build`
